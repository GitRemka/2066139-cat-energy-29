### Памятка по запуску проекта

#### 1. Выполнить команду для установки необходимых зависимостей: 
`npm ci` или `npm i` ;
#### 2. Выполнить команду: 
* `npm start` - для запуска проекта;
* `npm run build` - для сборки проекта.
